📅 分学期时间线

🔴 大一 · 下（现在 → 2026.7）— 打基础 + 进入状态

  • 期末复习：确保不挂科，GPA 是留学材料之一
  • Java 巩固：面向对象、集合、IO、异常处理，用 Java 写一个小项目（控制台图书/学生管理系统）
  • 日语节奏：每天 30 分钟，目标 7 月 JLPT N5 或 12 月 N4
  • 暑假任务：MySQL 入门 → CRUD、JOIN、索引，配合 Java 做 JDBC 连接练习

🟠 大二 · 上（2026.9–2027.1）— 前后端打通

  • Spring Boot 入门：Servlet → Spring MVC → Spring Boot，能写简单 REST API
  • 第一个完整项目：前后端联调的个人博客或 Todo 应用（Vue/React 前端 + Spring Boot 后端 + MySQL）
  • 数据结构：用 Java 实现常用结构（链表、栈、队列、二叉树），LeetCode 简单题 50+
  • 日语：12 月考 JLPT N4（底线 N5）

🟠 大二 · 下（2027.2–2027.7）— 深入后端 + N3

  • Spring 全家桶：MyBatis/MyBatis-Plus、Spring Security、Redis 缓存
  • Linux 基础：常用命令、部署 Spring Boot 到云服务器
  • 第二个项目：带登录鉴权的后台管理系统，上传 GitHub
  • 日语 N3：7 月冲 N3，这是关键节点
  • 开始调研日本大学：看目标校官网、教授研究室、研究方向

🟡 大三 · 上（2027.9–2028.1）— 冲刺 N2 + 准备研究

  • 日语 N2：12 月必须拿下（日本留学最低语言门槛），优先级最高
  • 微服务入门：Spring Cloud / Docker，理解容器化部署
  • 实习准备：完善简历（两个项目 + GitHub 绿格子），投递寒假实习
  • 确定研究方向：在软件工程/AI/数据方向中选一个感兴趣的，读相关论文
  • 写研究计划书初稿：800–1200 字日语，明确研究课题、方法、预期成果

🟡 大三 · 下（2028.2–2028.7）— 套瓷 + 实习

  • 寒假/暑假实习：至少一段开发岗实习经历（日本教授看重实践）
  • 联系教授（套瓷）：3 月开始给日本教授发邮件，附简历和研究计划书
  • 英语：部分学校要求托福/托业（目标托业 700+ 或托福 80+），视目标校而定
  • 日语口语：练习面试日语，准备常见问题回答
  • N1 冲刺：7 月尝试 N1（非必须但巨大加分）

🟢 大四 · 上（2028.9–2029.1）— 出愿 + 考试

  • 出愿材料：成绩单、毕业证明、推荐信（2封）、研究计划书、语言成绩
  • 参加入学考试：各校笔试（专业课 + 数学）和面试
  • 毕业设计开题：选与未来研究方向相关的题目

🟢 大四 · 下（2029.2–2029.6）— 等 offer + 毕业

  • 收到内诺：确认录取校，办理在留资格、签证
  • 完成毕业设计：答辩、拿学位证
  • 赴日准备：租房、机票、生活安排

🏫 大学院申请流程图

1. 选校选教授（大三上开始）
浏览大学官网 → 看教授研究方向 → 读 2-3 篇目标教授的论文 → 确认是否招研究生
2. 写研究计划书（大三上~下）
800–1200 字日语，包含：背景、目的、方法、预期成果、参考文献
3. 联系教授/套瓷（大三下 3月~）
发邮件 → 附简历+研究计划书+成绩单 → 约线上见面 → 争取内诺
4. 出愿（大四上 9–12月）
提交出愿材料 → 缴纳检定料 → 等准考证
5. 考试（大四上 10月~大四下 2月）
笔试（专业课）+ 面试（研究计划书答辩）→ 按学校时间
6. 拿内诺 → 办签证 → 赴日
确认录取 → 在留资格 → 签证 → 出发
